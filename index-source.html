POF Gym Management System

A professional, full-featured gym management platform built specifically for Pakistan Ordnance Factories. Manage members, track fee payments, configure shift schedules, and view detailed analytics — all in one elegant, modern interface.

🌐 Portfolio 💻 GitHub ✉️ Contact

Key Features
  • Secure manager login with session management
  • Full CRUD operations for gym members
  • Monthly fee tracking with visual history grid
  • 3-category member management (POF / Son of POF / Civilian)
  • Configurable shift timings per category
  • Analytics & reporting dashboard
  • Dark & light mode with seamless switching
  • Real-time search and advanced filtering
  • Responsive design for all screen sizes
  • Persistent local data storage
⚙️
Technology Stack

Built with pure, vanilla web technologies — no frameworks, no dependencies. Designed for maximum performance and zero reliance on external servers.
